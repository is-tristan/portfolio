# Tristan Wills - Fullstack Portfolio Website

Personal portfolio built with [Astro](https://astro.build) and Sass.

## Setup

```bash
npm install
```

## Scripts

- `npm run dev` — Start dev server
- `npm run build` — Build for production
- `npm run preview` — Preview production build

## Deploy

Deploys to Cloudflare Workers. Build, then:

```bash
npx wrangler deploy
```
